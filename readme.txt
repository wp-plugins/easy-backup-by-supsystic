== Description ==

Create Dropbox backup in one click. Easy-to-use functionality - now you don't have to spend few hours for learning plugin and its settings or looking for option that you need, because whether it is DropBox backup, local backup or backup restoration - every feature of DropBox backup plugin can be used from one page.

= DropBox Backup plugin features =

* Backup to DropBox cloud service
* Backup to FTP 
* WordPress website Restoration. Plugins, posts, database, themes and files
* Customisation of Backup. Plugins, WordPress Core, Database backups
* Backup in archive with .zip
* Backup Logs
* Backup restoration via DropBox or Local backup

== Installation ==

= To install a dropbox backup plugin via FTP, you must = 

1. Download the dropbox backup plugin
2. Unarchive the DropBox Backup by Supsystic  plugin 
3. Copy the DropBox Backup by Supsystic folder with plugin
4. Open ftp \wp-content\plugins\
5. Paste the plug-ins folder in the folder
6. Go to admin panel => open item "Plugins" => activate the plugin
